粤语转VOCALOID发音符号
======================

一个把粤语转换为VOCALOID发音符号的小程序，基于cccedict的粤语发音词典。

功能
----

- 把简体汉字转换成粤语拼音以及VOCALOID发音符号

原理
----

1. 把输入的汉字在词典中找出其粤语发音
2. 根据发音匹配发音符号
3. 制表输出

工作过程
--------

1. 输入文本后逐个调用转换函数。
   - 调用寻找函数找到该字的发音
   - 循环找出该发音的字母在不改变顺序下的所有组合
   - 每个组合都匹配一次发音符号表，留下表中含有的组合并获取其符号
   - 去重
   - 筛选掉短组合留下含有该短组合的长组合
   - 拼接组合得到符号
   - 将字、拼音以及符号加入表格
2. 输出表格
